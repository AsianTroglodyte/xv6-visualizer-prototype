xv6 JSLinux handoff

Serve xv6 assets here; run them through upstream JSLinux/TinyEMU

JSLinux Server QEMU
Open in JSLinux Config Kernel Disk

The config points JSLinux at assets/kernel and the TinyEMU HTTP block manifest generated from assets/fs.img. This server sends CORS headers so the upstream emulator can fetch those files from this origin.

Compatibility note: xv6-riscv is built for QEMU's virt machine. If TinyEMU/JSLinux does not boot it directly, use the server QEMU terminal.
